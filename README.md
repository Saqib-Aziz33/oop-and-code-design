# OOPs and SOLID Principles Refresher

A quick refresher on **Object-Oriented Programming (OOP)** and **SOLID principles** using **Python 3.11.9**.

Includes:

- Core OOP concepts (Encapsulation, Inheritance, Polymorphism, Abstraction)
- Best practices and comparisons with functional programming
- Solid Programming principles
- Curated learning resources
- `prompt.md` for **prompt engineering**, which can be fed into LLMs to accelerate learning
